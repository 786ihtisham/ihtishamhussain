How It Was Built

OrbitBrief is intentionally scoped as a safe public MVP: workflow and integration layer only, not a certified flight-dynamics or collision-avoidance engine.

Frontend

Angular/TypeScript dashboard concept with typed models, filters, risk badges, detail views, and review states.

Backend

Java/Spring Boot REST API design for satellites, daily briefing summaries, conjunction filtering, and review actions.

Product

Micro-SaaS case study under DataSourceCode, built to show domain learning, API thinking, and user-focused dashboard design.
