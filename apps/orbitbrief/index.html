Interview Story

How to explain it in 30 seconds.

OrbitBrief is a standalone dashboard MVP, not a certified collision-avoidance system. It demonstrates how I turn public domain data into an operator workflow: fetch multiple APIs, normalize signals, handle failures gracefully, calculate a useful product metric, and present a clear briefing UI.
Built by Ihtisham Hussain as a DataSourceCode side-project MVP. Public feeds: WhereTheISS, Launch Library 2, NASA APIs. Not for operational safety decisions.
